# 🚀 嘉兴骑手集市 - 快速启动指南

## 环境准备

1. 安装 Node.js (推荐版本 16.x 或更高)
   - 下载地址: https://nodejs.org/

## 启动步骤

### 第一步：初始化数据库

```bash
cd backend
npm install
npm run init-db
```

### 第二步：启动后端服务

```bash
# 在 backend 目录下
npm run dev
```

后端服务将在 http://localhost:3000 启动

### 第三步：启动前端服务

打开新的终端窗口：

```bash
cd frontend
npm install
npm run dev
```

前端服务将在 http://localhost:5173 启动

## 注意事项

1. 图片上传功能目前使用占位图片，正式使用需要配置图片存储服务
2. 支付功能建议先使用线下交易，后续可接入微信/支付宝
3. 建议先在小范围测试，收集骑手反馈后再推广

## 技术支持

如有问题，请检查：
1. Node.js 版本是否正确
2. 端口 3000 和 5173 是否被占用
3. 数据库文件是否正确生成
